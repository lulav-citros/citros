Getting Started: Query Data

First of all, let's have a quick look at the ways to query data using class CitrosDB.

Query Data​

To get access to a Citros database, create CitrosDB object:

>>> from citros import CitrosDB
>>> citros = CitrosDB()

This way CitrosDB is created with defaults parameters. To specify connection parameters, pass corresponding arguments:

>>> citros = CitrosDB()

Data is always queried for the exact topic; you also need to know the names of the batch and simulation. For example, to query all data for topic 'A' from the batch 'batch_name', created by simulation 'simulation_name':

>>> df = citros.simulation('simulation_name').batch('batch_name').topic('A').data()
>>> print(df)
Show the result:
sidridtimetopictypedata.x.x_1data.x.x_2data.x.x_3data.timedata.timedata.y
030105036927Aa-0.080-0.00217.700.30.3[2, 28, 45]
110312751159Aa0.0000.080154.4710.010.0[15, 41, 43]
....................................

The result is a DataFrame of the pandas package.

Batch consists of two parts: json-data column, and all other columns. To query exact json-objects, pass list with their labels to data(). For example, if the json-data column looks like:

data
{'x': {'x_1' : 1, 'x_2' : 12, 'x_3' : 70}, 'y': [5.0, 3.4, 10], 'height' : 12}
{'x': {'x_1' : 5, 'x_2' : 10, 'x_3' : 73}, 'y': [5.5, 6.7, 50], 'height' : 11}
...

to query 'x_1', 'x_2', 'height' and values from the first position of 'y' json-array, the following code may be used:

>>> df = citros.topic('A').data(['data.x.x_1', 'data.x.x_2', 'data.height', 'data.y[0]'])

Also, different constraints may be applied to query, see examples.
