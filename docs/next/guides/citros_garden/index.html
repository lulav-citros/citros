Run a CITROS Garden Project

Clone CITROS Garden Projectâ€‹

All the projects can be found here

To clone the repository:

git clone git@github.com:citros-garden/repository_name.git

Run Project in VScodeâ€‹

Open Project in VScode Dev Containerâ€‹

  1. Open the cloned repository in VScode:

    cd ~/project-name
    code .

  2. Open and run Docker.

  3. Open the project in VScode.

  4. Press on the bottom left corner icon.

  1. Choose Reopen in Container

  1. Open VScode terminal and make sure you are in the right location. 

    ros@docker-desktop:/workspaces/[Project Name]$

  2. Build the project

Build the projectâ€‹

$ colcon build
$ source install/local_setup.bash

Initializationâ€‹

$ citros init

As you can see, a lot more is happening when you initialize your repository while being logged in. We will not delve into all the details behind the scenes, but as always, feel free to roam through the CLI Documentation for further details.

The command above will run the simulation on your machine, and save all the results under .citros/data/[simulation_name]/[batch_name]/[date] folder.
The content of the folder will contain:

  • recorded bags
  • logs from the simulation and citros itself
  • metadata about the run
  • metrics and information about the system it was running
  • and more.

To view all simulations use the data command:

$ citros

Verify Data Loadedâ€‹

To verify the data is loaded

citros data list

Batch Statusâ€‹

  1. LOADED - the data is loaded to the database
  2. UNLOADED - the data is loaded to the database

Execute Notebookâ€‹

CITROS provides a citros data analysis package that gives access and functionality to analyze the data. We recommend to use this package with a python notebook which allows you to generate reports down the line.

note

citros data analysis package is installed while running pip install citros, so at this point you should have this package and you don't have to download it again.

So now, after we uploaded the data, we can execute the notebook

  1. Go to notebooks folder in your project.

  2. Open [notebook_name].ipynb

  3. Press Run All button

  4. Scroll down the notebook to see the results.
