citros data db logs

Description​

the logs command prints the logs of the database instance.

$ citros data db logs [-h] [-d] [-v]

Options​

OptionDescription
-h, --helpShow help message
-d, --debugSet logging level to debug
-v, --verboseUse verbose console prints

Example​
