Poliastro

Overview​

This project contains three simple examples using Poliastro lib for Python3:

  1. Simple orbit example
  2. Orbital maneuver example
  3. Atmospheric drag simulation.

You can find all the information about the used functions and the mathematical explanation on the Poliastro Website.
All project installation, code overview and usage details are also available on the project's GitHub page.

Prerequisites​

  1. Please make sure you have all the necessary softwares to work with CITROS installed on your computer.
  2. Install Visual Studio code.
  3. We strongly recommend that you work with dockers. However, if you wish to work without dockers, please refer to the .devcontainer directory in project's repo, the dependencies you need are in the Dockerfile file.
  4. (Optional) Install Foxglove.

Scenario​

The Poliastro project consists of three packages suitable for orbital mechanics calculations:

  • Poliastro_simple_orbit
    This package is used to calculate vessel orbital coordinates around the Earth between time bounds from given apoapsis and periapsis altitudes.
    The result is an ephemerids of orbit (actually a part of it between given time bounds) with zero right ascension of the ascending node, argument of the pericenter and true anomaly for simplicity.
  • Poliastro_maneuver
    This package calculates three orbits for the Hohmann transition: an initial orbit, an intermediate orbit, and a final orbit.
    The simulation takes the radius of the initial orbit and the radius of the final orbit as input.
    The result is the ephemerids of these orbits, not the trajectory!
  • Poliastro_atmo_drag
    This package shows the effect of aerodynamic drag forces on an artificial satellite in low Earth orbit.
    The simulation takes Earth diameter, drag coefficient, Keppler orbit parameters and maximum simulation time as inputs.
    The result is a plot of altitude versus time and flight time before hitting the surface.

Drag forces calculations are more difficult and more rare than the others, hence, in this tutorial we will use the Atmospheric Drag simulation scenario.
Let's check how the exact orbit duration reduces depending on Drag coefficient. For that, we will set up Drag Coefficient (c_d) parameter random distribution in range from 2 to 4.
The parameter will be set by a NumPy random function and its setup is listed in .citros/parameter_setups/default_param_setup.json:

    "packages": {
        "poliastro_atmo_drag": {
            "poliastro_atmo_drag": {
                "ros__parameters": {
                    "earth_r": 6378.1366,
                    "a": 6550.85876692,
                    "ecc": 0.0,
                    "inc": 0.0,
                    "raan": 0.0,
                    "argp": 0.0,
                    "nu": 0.0,
                    "c_d": {
                        "function": "numpy.random.uniform",
                        "args": [2, 4]
                    },
                    "t_limit": 1.0,
                    "publish_freq": 10.0
                }
            }
        },

Learn more about parameter setup and defining custom functions in Directory parameter_setups and Adding Functions to Parameter Setup pages.

In addition to parameter setup, you can configure the simulation performance setup (timeout, CPU, GPU and Memory) as well. This parameters can be found in .citros/simulations/simulation_poliastro_atmo_drag.json.
Look in Directory simulations page for more information.

Running the Scenario Using CITROS​

Running Locally​

Follow these steps to Run Simulation:

  1. Name your batch run simulation: poliastro
  2. Add a message to your batch run simulation: local test run
  3. Run your simulation 5 times .

All the results will be saved under .citros/data/simulation_poliastro folder.

To plot the local run results you can use Foxglove with the layout file existing in foxglove_layout folder.

Results​

  1. Create Database
  2. Upload data to the database
  3. Verify the data was loaded
  4. Execute the Notebook pliastro_notebook_example.ipynb.
    You will find the notebook under citros_template/notebooks folder.
